【實戰救援】Excel 與建築工程報表的精準聯動

#Excel自動化 #工程報表救星 #防呆自動校驗 #小白實戰手冊 #岫校長軍火庫

👥 本教學適用對象與用途

適合人群：

  • 現場估價與工地人員：厭倦了手動加總、擔心報價單少算漏算賠錢的人。
  • 行政與採購助理：每天花大量時間在 Excel 修改單價，整本報表卻越改越亂的初學者。

這套系統能幫你解決什麼問題？(使用場景)：

  • 變更設計時的救命符：當業主臨時修改設計或材料價格變動時，你不用重新算，報表自動同步，保證獲利不縮水。
  • 叫料漏項的防護網：透過智慧警衛，只要某項材料忘記叫料、或者數量為 0，系統立刻跳出紅框警告，防止工地停工。
  • 報表交差的零失誤率：確保送給辦公室或業主的數據，已經過系統校驗，絕對不會因為打錯數字而丟臉或賠錢。

本教學用途：

  • 將「人肉計算」轉化為「自動化聯動」，徹底消除人為疏失。
  • 建立一套「防呆機制」，確保報表在送出前就能抓出錯誤。

🛠️ 保姆級教學：連電腦小白都能上手

第一步：建立你的「數位總倉」（小白必做）

很多小白報表做不起來，是因為把所有資訊全塞在同一頁。請開一個新分頁叫「材料資料庫」，A 欄寫材料名稱，B 欄寫單價。以後改單價，全報表自動同步，不用再按計算機！

第二步：用 VLOOKUP 綁定報表 (神功)

在金額欄位輸入：=VLOOKUP(搜尋關鍵字, 材料資料庫!範圍, 2, 0)。這是在告訴 Excel：「自動去總倉查單價，填過來」，讓你永遠不會算錯數字。

第三步：自動校驗（岫老師的防呆機制）

使用「條件式格式設定」，只要金額欄位出現「負數」或「空白」，就讓格子變紅。這就是你的保命符，亮紅燈馬上修，千萬不要送出報表。

⚠️ 緊急救援：Excel 報表炸裂時怎麼救？

如果不小心把公式刪掉、或者報表全變成了 #REF! 亂碼：

  • 不要慌，按 Ctrl+Z：先還原上一動。
  • 檢查參照源：點選報表中的公式格子，看看 Excel 框出的參考範圍是不是指向了錯誤的頁面。
  • 重新連結：只要把公式重新指向正確的「材料資料庫」位置，一切就會恢復原狀。

⚔️ 軍火庫 (自動化警衛代碼)

這是岫老師寫好的「智慧警衛」VBA 腳本，放進去，報表只要有漏項，Excel 會自動跳出視窗警告你：

Sub AutoCheck報表()
If Range("C10").Value = 0 Then
MsgBox "注意！報表有金額為 0 的欄位，請檢查是否漏算！", vbCritical
End If
End Sub

🛡️ 岫校長保姆級戰術：Excel 的「智慧警衛」 (VBA 教學)

很多新手做報表，最怕「漏填一個數字」。這個功能幫你 24 小時盯著報表，只要少寫一個，它立刻跳出來罵你。

第一步：開啟「開發人員」控制台
對著選單點右鍵選擇「自訂功能區」，勾選「開發人員」。

第二步：把警衛放進去
進入「開發人員」>「Visual Basic」>「插入」>「模組」，貼上以下代碼：

Sub AutoCheck警衛()
If Range("C10").Value = 0 Then
MsgBox "岫校長警告：這份報表漏掉了重要金額，請檢查 C10 欄位！", vbCritical, "報表檢查警衛"
Else
MsgBox "檢查完成：目前報表金額正常，可以放心送出！", vbInformation, "報表檢查警衛"
End If
End Sub

第三步：測試警衛
回到 Excel >「巨集」> 點選 `AutoCheck警衛` >「執行」。若漏填，會跳出紅框警告；填寫正確，則會提示安全送出。
